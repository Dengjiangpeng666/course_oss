# 前端 · 面向专业认证的毕业要求达成度统一计算平台

本目录为平台的前端工程，采用 Vue 3 + Vite 构建，负责提供专业认证数据采集、配置与分析的 Web 可视化界面。

## 技术栈

- Vue 3
- Vite
- （如已安装可补充 UI 框架，例如 Element Plus、Ant Design Vue 等）

## 启动方法

```bash
npm install
npm run dev
```

开发过程中，前端会通过 RESTful 接口与后端 FastAPI 通信。
后端默认地址为 `http://127.0.0.1:8000`（如有更改请同步修改前端 API 配置）。

## 主要功能规划（简要）

- 角色登录与权限管理
- 数据字典、课程与学生名单批量导入
- 考核点/课程目标等多层级配置表单
- 达成度计算数据展示、报表导出

> 当前阶段为前端初始化与联调，详细功能开发将按计划逐步推进。
